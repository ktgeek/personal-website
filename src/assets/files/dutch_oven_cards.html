🖨️ Print Instructions — open this file in Chrome

  1. Open in Google Chrome (Safari does not handle print CSS reliably for this layout).
  2. Press ⌘P to open Print.
  3. Set Destination to your printer.
  4. Set Paper size to US Letter.
  5. Set Margins to None.
  6. Set Scale to 100% — do not use "Fit to page".
  7. Turn on Background graphics (in "More settings") so the colored table headers print.
  8. Use duplex printing to print the page or
    1. Print Page 1 only first (fronts). Let the sheet fully exit the printer.
    2. Flip the sheet on its long edge (top goes to bottom), reload into the tray face-down.
    3. Print Page 2 only (backs). Cut along the dashed lines.

You get 8 double-sided cards per sheet.

⚠️ The cards are centered on the sheet with ~0.75" side margins and ~0.875" top/bottom margins. The 0.5" gap between cards provides generous lamination overlap.

Dutch Oven Charcoal Tips:
Start charcoal briquettes about 15–20 minutes before needed
Use a chimney to light — it’s so much easier for lighting charcoal!!
Arrange proper number of coals under the Dutch Oven in a circle, about ½” inside the oven’s edge
Arrange briquettes on top lid in a checkerboard pattern
Rotate oven ¼ turn every 10–15 min for even cooking — reduces hotspots & slow cooking from cool wind
